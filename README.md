thenaterhood.github.io
======================

Personal website of Nate Levesque (thenaterhood).

Local Setup
----------------------
Requirements:

* Ruby
* jekyll
* jekyll-paginate
* jekyll-sitemap

Once you have Ruby, the Jekyll packages are installable as gems.
